Python3安装支持库的过程中经常会遇到“Microsoft Visual C++ 14.0 is required”，此时就需要安装“Visual C++ build tools”生成工具。而新版的”Microsoft Visual C++ “ 十分臃肿，占用空间巨大，所以就分享这一旧版“Microsoft Visual C++”，解决此类问题。

## 下载

<https://pan.baidu.com/s/1NCCckatHwxOPAxzp1WgOkw?pwd=67es>

## 安装

1. 打开“solved_packageMissInInstalling_mu_visual_cpp_build_tools_2015_update_3_x64_dvd_dfd9a39c.iso”

![057d0d80-e630-4bdc-9bda-1e7c5fc29dda.png](/post/057d0d80-e630-4bdc-9bda-1e7c5fc29dda.png)

2. 运行“Visual Cpp Build Tools_Full.exe”

![c529f9b9-e7a6-4709-bb1f-591a09ea27ce.png](/post/c529f9b9-e7a6-4709-bb1f-591a09ea27ce.png)

3. 选择“默认值”，点击安装
